# Chess Duel

Chess Duel is a complete chess game built with Unity 6.

## Features

- Single Player AI
- Online Multiplayer (Photon Fusion)
- Chess Clock
- Castling
- En Passant
- Pawn Promotion
- Check
- Checkmate
- Stalemate
- Animated UI

## Built With

- Unity 6
- C#
- Photon Fusion
- TextMeshPro
